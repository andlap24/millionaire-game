### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher
* NPM classNames


### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm install classnames --save`
* Run `npm install react-burger-menu --save`
* Run `npm start`
